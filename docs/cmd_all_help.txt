Command Line Interface Help Documentation
=====================================


acpi
====
usage: acpi [-h] {list,show} ...

ACPI table management tool

positional arguments:
  {list,show}  Commands
    list       List all ACPI tables
    show       Show specific ACPI table content

options:
  -h, --help   show this help message and exit

CPU information and management tool

  {info,temp,freq,usage,topology,governor}
                        Commands

pci
===
usage: pci [-h] [-v] [-t TYPE] [-b BUS]

List PCI devices in the system

options:
  -h, --help            show this help message and exit
  -v, --verbose         Show detailed information
  -t TYPE, --type TYPE  Filter by device type (e.g., "VGA", "Network", "Storage")
  -b BUS, --bus BUS     Filter by bus address
