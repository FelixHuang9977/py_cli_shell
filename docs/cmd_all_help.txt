Command Line Interface Help Documentation
=====================================


info
====
Built-in command for showing command information

Usage: info [-a] [-p]
  -a, --all   Show full path and description
  -p, --path  Show command path

marko
=====
usage: marko [-h] [{usb,cpu,all}]

Run diagnostic test cases

positional arguments:
  {usb,cpu,all}  Test category: usb, cpu, or all

options:
  -h, --help     show this help message and exit

pci
===
usage: pci [-h] [-v] [-t TYPE] [-b BUS]

List PCI devices in the system

options:
  -h, --help            show this help message and exit
  -v, --verbose         Show detailed information
  -t TYPE, --type TYPE  Filter by device type (e.g., "VGA", "Network",
                        "Storage")
  -b BUS, --bus BUS     Filter by bus address
